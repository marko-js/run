# LLM docs

Files optimized for feeding to coding models/agents working with @marko/run:

- `marko-run-cheat sheet.md` — dense prompt-sized sheet (routing, handlers,
  middleware, `$global` context, redirects).
- `marko-run-reference.md` — the comprehensive reference it distills.

Both were validated together with the Marko 6 sheets by a controlled
experiment (weak-model subjects, deterministic graders): see
`marko-js/marko` → `llm/experiment/` for protocol, tasks, and results.
